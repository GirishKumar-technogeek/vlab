Experiment to interface with IoT Kit to demonstrate the Working of LED,SWITCH and BUZZER

Aim

To demonstrate the working of LED, SWITCH and BUZZER by interfacing with IoT kit.


Procedure
  • This simulator allows users to simulate interfacing of LED,Buzzer,Switch with Raspberry Pi board.
  • In the Experiment tab, you can get to know the aim of this experiment,how to work with this simulator and a code reference enabling you to program the raspberry pi board.
  • Using the given code below, program the raspberry pi board.
  • Once you finished programming the board,click on the check code button.If you get a success message you can proceed for the simulation, Else please take a look at the code and reprogram the board.
  • In the simulation part, you can connect the LED,buzzer,push button,ground connections with a click of button.
  • Click on the Connect LED,Connect Buzzer,Connect Button,Connect Gnd Connections buttons to connect the components with GPIO pins.
  • Once you connect all the components, click the simulate button which will check the correctness of the circuit and gives you the result as alert.
  • Once the simulation is successful click on the final simulation tab.
  • Click the Push button and you can note that LED and buzzer is in ON state for the time interval you mentioned during the programming step.

Code
                
                  import time,RPi.GPIO as GPIO
                  GPIO.setmode(GPIO.BOARD)
                  GPIO.setup(11, GPIO.OUT) #LED
                  GPIO.setup(13,GPIO.OUT) #BUZZER
                  GPIO.setup(23, GPIO.IN, pull_up_down=GPIO.PUD_UP)
                  try:
                    while True:
                      button_state = GPIO.input(23)
                      if button_state == False:
                        GPIO.output(11, True)
                        GPIO.output(13, True)
                        print('Button Pressed...')
                        time.sleep(0.2)
                      else:
                        GPIO.output(13, False)
                        GPIO.output(11, False)
                        print('Button Released...')
                  except:
                    GPIO.cleanup()
